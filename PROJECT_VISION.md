# Fashion Shop - Магазин одежи

## Опис проекту

Fashion Shop - це сучасний онлайн-магазин одежи, побудований на Django. Проект надає зручний інтерфейс для перегляду каталогу одежи, фільтрування по категоріям та розглядання детальної інформації про кожен товар.

## Основні функції

- **Каталог товарів**: Повний каталог одежи з фотографіями, описом та ціною
- **Категоризація**: Товари розділені на категорії (сорочки, штани, куртки, сукні)
- **Адаптивний дизайн**: Сайт добре виглядає на різних пристроях (мобільні, планшети, комп'ютери)
- **Чистий інтерфейс**: Використання Bootstrap для сучасного та професійного вигляду

## Технологічний стек

- **Backend**: Django 6.0
- **Frontend**: Bootstrap 5, HTML5, CSS3
- **Database**: SQLite (за замовчуванням)

## Структура моделей

### Category
- name: назва категорії
- description: опис категорії
- slug: URL-friendly ідентифікатор

### Product
- name: назва товару
- description: опис товару
- price: ціна товару
- image: зображення товару
- category: посилання на категорію
- created_at: дата створення
- updated_at: дата останнього оновлення
